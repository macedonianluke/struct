# VISIONARY WEB

The Next.js port of the Visionary "Anti-Agent" Real Estate App.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Architecture

-   **Frontend**: Next.js 14+ (App Router).
-   **Styling**: Global CSS (Brutalist System) in `src/app/globals.css`.
-   **Mock Data**: `src/lib/nanoBanana.ts`.
-   **Components**: `src/components/PropertyCard.tsx`.

## Key Features

-   **Brutalist UI**: High contrast, sharp edges.
-   **Nano Banana Vision**: Mocked AI architectural analysis.
-   **Interactive Slider**: Before/After comparison.
